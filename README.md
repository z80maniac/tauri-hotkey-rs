# Tauri Hotkey

This crate provides cross platform APIs to register keyboard hotkeys. This is a fork of [hotkey-rs](https://github.com/gamebooster/soundboard/tree/master/extern/hotkey-rs).

## Platform support

- Linux
- macOS
- Windows
